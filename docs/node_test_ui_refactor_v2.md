# 节点测试界面重构 V2

**日期**：2026-06-21  
**状态**：代码完成，待测试

---

## 变更概述

基于第一轮聊天界面改造，进行第二轮界面重构，参考用户提供的截图设计，实现更简洁的交互体验。

---

## 核心改动

### 1. 布局调整

**移除左侧边栏**
- 原：280px 宽固定左侧边栏，显示模式切换器和节点列表
- 新：取消左侧边栏，主内容区全屏展示聊天内容

**底部展开菜单**
- 模式/节点选择移到输入框底部
- 点击左下角按钮向上展开菜单
- 菜单内容：
  - 测试模式选择器（文本推理 / 图片生成）
  - 节点分组列表（保持原有分组逻辑）
- 选中节点后自动关闭菜单

### 2. 图片预览优化

**显示位置**
- 从输入框底部移到输入框上方
- 独立的预览区域，带边框和背景色

**预览功能**
- 使用 Ant Design `Image` 组件
- 点击图片可查看大图（内置预览功能）
- 删除按钮改为右上角圆形悬浮按钮
  - 位置：`top: -6px, right: -6px`
  - 样式：黑色半透明背景 `rgba(0,0,0,0.8)`
  - 圆形按钮，白色 × 图标

### 3. 输入区域重构

**整体布局**
```
┌────┬──────────────────────┬────────┐
│按钮│                      │        │
│组  │   文本输入框         │ 发送   │
│    │   (3行高度)          │ 按钮   │
│    │                      │        │
└────┴──────────────────────┴────────┘
```

**左侧按钮组**（垂直排列，无圆角）
- 模式/节点选择按钮
  - 图标：`<MessageOutlined />`
  - 尺寸：48px × 48px
  - 激活状态：蓝色背景 `token.colorPrimaryBg`
  - Tooltip：「选择测试模式和节点」
- 图片上传按钮（仅多模态/图生图时显示）
  - 图标：`<FileImageOutlined />`
  - 尺寸：48px × 48px
  - Tooltip：「添加图片」

**文本输入框**
- Flex 填充剩余空间
- 高度：3 行（`rows={3}`）
- 无边框、无 padding
- 字体：14px

**发送按钮**
- 尺寸：96px(高) × 80px(宽)
- 颜色：橙色 `#FF6B35`（主题色）
- 图标：`<SendOutlined />`（18px）
- 文字：「发送」（14px，500 粗细）
- 禁用状态：灰色背景

**整体样式**
- 所有按钮无圆角 (`borderRadius: 0`)
- 按钮间无间隙，紧密拼接
- 左侧按钮组用 `1px solid border` 分隔

### 4. 交互优化

**底部菜单**
- 点击模式/节点按钮：切换展开/收起状态
- 点击节点：选中节点 + 自动关闭菜单
- 菜单高度：最大 400px，超出滚动
- 节点列表高度：最大 300px，超出滚动

**状态持久化**
- 节点分组折叠状态：`settings.json` → `nodeGroupExpanded`
- 测试历史：`settings.json` → `testHistory`
- 节点配置：`settings.json` → `nodeTestFormPerNode`
