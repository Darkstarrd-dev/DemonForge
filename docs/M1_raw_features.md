- 文件来源：盗版渠道二次分发（「灵珑小说群」系 QQ 群 + ctfile 网盘，见 §3.1/3.2）。

## 4. 正文穿插乱码样例

> **混淆机制分析（2026-06-12，用户提供关键洞察）**：这些混淆不是给人眼看的，而是
> **针对 TTS（听书）的**——人眼会自动跳过杂讯并脑内复原句子，TTS 却会把谐音字逐字
> 念出来，让听书用户"听到"群号；零宽符等不可见字符则是"不会被念出来"的填充干扰。
> 行首竖排藏号（§4.3）对人眼无效、对 TTS 跨行拼读也几乎无效，属低效变种。
> 该洞察是清理 prompt v2 与规则清理引擎的设计出发点（`M1_text_cleaning.md` §3.7/§3.9）；
> 推论「群号必然在文件内以明文出现（否则广告无效）」即规则引擎的自锚定原理。

按混淆手法分五类。共性：**母体是 §3.2 的群号**，被拆成碎片塞进正文。

### 4.1 数字碎片直插

```
　　“明3717天开始，晚饭后练习一小时。”29119
　　“毛小説羣89利兰都出现了，还有什么动漫39644女主60会出现？”
　　下一秒，他出现灵珑中转在一群893辆装甲964车旁。460
　　优美小説羣817子040看着545她。
```

特征：群号拆成 2–5 段顺序插入；**常把词语、人名从中间劈开**
（毛利兰 → 毛[小説羣89]利兰；优美子 → 优美[小説羣817]子；公平竞争 → 公[…]平竞[…]争），
删除碎片后需复原被劈开的词语。

### 4.2 变体字 + 多写法数字混淆

```
　　宭   893jiu6肆饲⒍零                        ← 拼音+谐音+带圈数字 = 893964460
　　“会仦裞羣3七的1七29。”11九                  ← 仦裞羣=小说群；3七1七29+11九=371729119
　　“怪物.中转羣..三七一狼人？”目暮警部瞪大眼七二九一一九睛。
　　阳乃站微笑着起身，“中转峮公  3气1漆平竞（二）9吆伊9争，那就...合作愉快？”
```

样本归纳的混淆字符表（清理 prompt 特征规则的依据）：

| 类别 | 样本中出现的写法 |
|---|---|
| 「群」变体 | 羣 峮 裙 囷 箘 宭 |
| 「说/小」变体 | 説 裞 仦 |
| 数字：标准写法 | 阿拉伯 `3`、全角 `８`、汉字小写 `三 七 九`、汉字大写 `壹 贰 弍 叁 肆 陆 柒 玖` |
| 数字：谐音/借形（按 0–9） | 0: 零 〇 邻 ⊙；1: 幺 吆 伊 壹 衣 医 易；2: 贰 弍；3: 叁 伞 彡；4: 肆 似 寺 泗 饲 私 斯 死；5: 伍；6: 陆 柳 遛 榴 硫；7: 柒 漆 气；8: 扒 疤 紦 吧 爸 芭；9: 玖 久 酒 究 韭 疚 镹 |
| 数字：符号形 | 带圈 `②④⑤⑦⒍`、带括号 `（一）（二）（八）`、罗马数字 `IX` |
| 字母/拼音 | `jiu si liu ba z b u n`、西里尔 `х`、数学字母 `𝚥 𝘬`（非 BMP，raw 176 行） |
| 拆字 | `亻尔`（=你，raw 216 行）、`衣 衤` 穿插（raw 264 行） |

> 该表为样本归纳，必然不完备。清理 prompt 应描述**模式**（「任意写法的数字序列 + 群/说类
> 变体字即广告碎片」），而非穷举字符。规则引擎按**拼音家族**泛化（读音∈ ling/yi/er/san/
> si/wu/liu/qi/ba/jiu 的字均可作数字证据），原型实装为强/弱两级映射表
> （`frontend/src/utils/ruleClean.ts` 资产层，可按新来源扩充）。


特征：同一人名反复损坏为「拔桤灸×」开头的汉字串，且损坏延伸到其后整个短句——
**这些字符串占位着原文**（典型 GBK/UTF-8 错位解码形态），机械删除即丢失句子内容，
也无法用正则修复。

### 穿插位置规律

- 无固定位置：段首、句中（含词语/人名内部）、段尾、独立成行、连续行首（竖排）均出现；
  章节标题行同样会被穿插（§2 末例）。
- 密度差异极大：从整段一两处到几乎逐词（§4.4）。
- 同一文件内多种混淆手法并存。

## 5. 格式问题

- 空行：段间 1 空行为主；章界与广告块附近出现 8 行以上连续空行（raw 30–38 行）。
- 段首缩进：全角空格×2 为主；部分段落为半角空格×2（raw 131–156 行）；
  mojibake 样例段无缩进（raw 267 行起）——同一文件内三种缩进混杂。
- 其他：全角/半角省略号混用（`……` 与 `...`）；半角冒号（`沉吟片刻:`）；
  全角数字（`８`）；行中全角空格作分隔。

## 6. 容易误删的内容（清理时必须保留/谨慎）

- 半角省略号 `...`、破折号、纯标点台词与拟声词（`“…！”`、`“滋啦——！！”`、`痛痛痛痛！`）
  ——作者文风，勿当杂讯清掉。
- 正文中的真实数字：`一千八百万次挥刀`、`十六岁`、`明天8k` 等，与群号碎片只能靠语义区分。
- 括号旁白/吐槽是网文常见文风，仅平台水印括注（`（看暴爽小说，就上飞卢小说网！）`）应删。
- 作者 ps/请假/求票（§3.3）：作者本人所写 → 去留待拍板（DESIGN.md §7 问题 7），拍板前保留。
- mojibake 段（§4.5）：表面乱码、实为损坏的原文 → 策略待拍板（§7 问题 6），拍板前保留。
