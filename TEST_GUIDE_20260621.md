## 预期结果

- 图片模式：中央展示图片，底部历史缩略图

## 已知限制

1. **TypeScript 类型警告**：部分旧代码有类型警告，不影响运行
2. **测试历史图片**：图生图/多模态的输入图片无法从历史恢复（仅记录 URL）

## 问题排查

### 节点测试页面崩溃
- 检查浏览器控制台错误
- 确认 `nodeGroupExpanded` 字段已添加到 store
- 重置 settings.json 后重试

### 分组折叠状态不保存
- 检查 `server/src/data/settings.json` 是否有 `nodeGroupExpanded` 字段
- 查看浏览器控制台是否有持久化错误

### Tab 不固定
- 检查浏览器 CSS 渲染
- 确认 `tabBarStyle` 样式正确应用
